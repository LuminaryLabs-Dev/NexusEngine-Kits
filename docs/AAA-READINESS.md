# AAA Readiness

This repo is not yet a complete AAA game stack.

It is a clean foundation for building toward one through official kits, domains, bundles, adapters, tests, and parity tracking.

## Current state

```txt
installer: scaffolded
catalogs: scaffolded
contracts: scaffolded
first-wave kit docs: in progress
real behavior parity: incomplete
AAA vertical slice: not complete
```

## Required before AAA work

- Real first-wave kit behavior.
- Real domain smoke tests.
- Snapshot and reset behavior for stateful kits.
- Asset, performance, and render descriptor discipline.
- CDN and package import stability.
- Tagged releases.
- A working vertical slice built from official kits.

## First vertical slice

The first proof should combine input, spatial interaction, progression, camera feedback, render descriptors, and one game-family domain.
