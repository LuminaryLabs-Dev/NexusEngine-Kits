# Source Parity

Stable source: NexusEngine-ProtoKits commit `e4e8a1e03943d1fb7ab1bfddb8837ad16b919c29`.

Preserved behavior includes repository, kit, domain, and bundle manifests; trusted-owner metadata; strict ID, domain-path, and API collisions; lookup, search, indexes, progress, reset, and exact snapshots. Older ProtoKit registry and manifest owners were merged into this canonical service.

Transport, Git resolution, caching, integrity verification, lockfiles, module execution, and installation remain separate adapter concerns by design.
