# Foundation Domain

The foundation domain contains low-level support kits used by other domains.

## Member kit themes

- seeded values
- clocks and timers
- state digests
- replay helpers
- health reports
- budget signals

## Install

```js
import { createFoundationDomainKits } from "@luminarylabs/nexusengine-kits/domain-foundation";
```

## Status

Scaffolded. Behavior parity with ProtoKits is not complete yet.
